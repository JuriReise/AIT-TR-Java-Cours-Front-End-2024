Flexbox

Flexbox (или CSS Flexible Box Layout) – это мощный инструмент CSS, который упрощает создание гибких и адаптивных макетов. С его помощью можно управлять расположением, выравниванием и распределением элементов внутри контейнера. Рассмотрим ключевые свойства Flexbox на основе предоставленного примера кода.
Основные понятия

    Flex-контейнер: элемент, на который применён display: flex;. Он управляет дочерними элементами, называемыми flex-элементами.
    Flex-элементы: дочерние элементы flex-контейнера.

Свойства Flexbox
Свойства для Flex-контейнера

    display: flex; Делает элемент flex-контейнером. Все его дочерние элементы становятся flex-элементами.

    flex-direction Управляет направлением главной оси (по которой выстраиваются flex-элементы).
        row (по умолчанию): элементы располагаются в строку слева направо.
        row-reverse: элементы располагаются в строку справа налево.
        column: элементы располагаются в столбец сверху вниз.
        column-reverse: элементы располагаются в столбец снизу вверх.

    Пример:

    flex-direction: column;

justify-content Управляет выравниванием элементов вдоль главной оси.

    flex-start (по умолчанию): элементы выравниваются в начале оси.
    flex-end: элементы выравниваются в конце оси.
    center: элементы выравниваются по центру оси.
    space-between: равномерно распределяет элементы, оставляя промежутки между ними.
    space-around: равномерно распределяет элементы с равными отступами вокруг.
    space-evenly: равномерное распределение с одинаковыми отступами между элементами и краями.

Пример:

justify-content: center;

align-items Управляет выравниванием элементов вдоль поперечной оси.

    stretch (по умолчанию): элементы растягиваются, занимая доступное пространство.
    flex-start: элементы выравниваются у начала оси.
    flex-end: элементы выравниваются у конца оси.
    center: элементы выравниваются по центру оси.
    baseline: элементы выравниваются по их базовой линии текста.

Пример:

align-items: start;

flex-wrap Указывает, нужно ли переносить элементы на новую строку или столбец, если они не помещаются.

    nowrap (по умолчанию): все элементы остаются на одной строке.
    wrap: элементы переносятся на новую строку.
    wrap-reverse: перенос элементов, но в обратном порядке.

Пример:

flex-wrap: wrap;

align-content Управляет выравниванием строк flex-элементов (когда используется flex-wrap) вдоль поперечной оси.

    stretch (по умолчанию): строки растягиваются, чтобы заполнить контейнер.
    flex-start: строки выравниваются у начала оси.
    flex-end: строки выравниваются у конца оси.
    center: строки выравниваются по центру.
    space-between: равномерно распределяет строки с промежутками между ними.
    space-around: равномерно распределяет строки с равными отступами вокруг.

Пример:

align-content: center;

Свойства для Flex-элементов

    align-self Управляет выравниванием конкретного flex-элемента по поперечной оси, переопределяя align-items контейнера.
        Возможные значения: auto, flex-start, flex-end, center, baseline, stretch.

    Пример:

    align-self: center;

flex Короткая запись для управления размером flex-элемента. Включает три свойства:

    flex-grow: определяет, сколько свободного пространства элемент может занять.
    flex-shrink: определяет, насколько элемент может сжиматься.
    flex-basis: начальный размер элемента до распределения свободного пространства.

Пример:

flex: 1 1 auto;

order Управляет порядком элементов в flex-контейнере. Значение по умолчанию – 0. Элементы с меньшим значением отображаются раньше.

Пример:

order: 2;

Пример на основе урока

    body Flexbox используется для вертикального центрирования контента:

    display: flex;
    flex-direction: column;
    align-items: center;

.animal-gallery

    Элементы расположены в строку с переносом (благодаря flex-wrap: wrap).
    Элементы выровнены по началу поперечной оси (align-items: start).
    Распределение элементов по главной оси – центрирование (justify-content: center).

.animal-card

    Элементы расположены вертикально (flex-direction: column).
    Внутреннее выравнивание:
        По горизонтали (поперечная ось) – по центру (align-items: center).
        По вертикали (главная ось) – в начале (justify-content: start).
